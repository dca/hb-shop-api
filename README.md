
## Description

The backend apis for hbshop

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API Documents 

Run and Visit `http://localhost:3000/doc/`  

## System Documents

```bash
$ npm run compodoc
$ // Visit `http://localhost:8080/`  
```
